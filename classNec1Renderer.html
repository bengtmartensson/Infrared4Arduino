Detailed Description

A static class consisting of two functions that generate IrSignal-s from the NEC1 protocol parameters.

The NEC1 protocol is given in IRP notation as {38.4k,564}<1,-1|1,-3>(16,-8,D:8,S:8,F:8,~F:8,1,^108m,(16,-4,1,^108m)*) [D:0..255,S:0..255=255-D,F:0..255]
